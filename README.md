# Diabetes Risk Prediction

Predicting diabetes risk level (Low Risk, Prediabetes, High Risk) from 
routine health and lifestyle data using machine learning.

## Dataset
Kaggle — 6,000 patients, 19 features including BMI, fasting glucose, 
HbA1c, sleep, and stress levels.

## Models
- Logistic Regression: 94.8% accuracy, 93.6% F1
- Random Forest: 91.8% accuracy, 90.0% F1

## Key Finding
BMI and waist circumference are the strongest predictors of diabetes risk.

## How to run
Open `DataQuest.ipynb` in Google Colab and run all cells.
